# Node-Backend

## Prerequisities

- Node.
- NPM.

## Getting started

Install dependencies

```
npm install
```

-Update MONGODB_URI:key in config.js with your MongoDB URL.

-You can also udpate SECRET:key as required(optional).

Finally to start server run command:

```
npm run dev
```

Voila you are ready to GO!
